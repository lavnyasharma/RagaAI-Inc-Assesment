# MedDash — Quick Start Guide (readmore.md)

Welcome to MedDash! This project is a comprehensive Healthcare SaaS Dashboard.

## Key Links
- **Documentation**: See `README.md` for full technical details.
- **Environment Setup**: Copy `.env.example` (or use the provided `.env`) to the root.

## Architecture
- **Frontend**: Vite + React + TypeScript.
- **Backend**: Firebase (Auth, Firestore, Cloud Messaging).
- **Styling**: Modern Vanilla CSS.

## Setup
1. `npm install`
2. `npm run dev`

See the `README.md` for the full feature list and architecture overview.
